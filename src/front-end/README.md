> 🠋 Executing this terminal command initializes a new Node.js project by generating a "package.json" file.

    npm init -y

> 🠋 These constitute the primary packages employed in this project.

    npm install gulp
    npm install sass
    npm install jquery
    npm install bcrypt
    npm install express
    npm install mongodb
    npm install bootstrap
